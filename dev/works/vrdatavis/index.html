Context

This research project was aimed to visualize higher-dimensional data scaled to 3D in VR. We developed an ego-centric data-visualization technique using JS in VR for interaction and navigation across webspaces.

Research Lab

Research Focus

Build a tool to view and navigate search results in VR

Representing data in VR

Reducing higher-dimensional data with force-field implementation where force between nodes is scaled to their affinity.

Radial replusion force

A new force was introduce to avoid object oclusions in VR. Users should be able to see all the data points clearly in space.

Implementation

Force-simulated VR Visualization
